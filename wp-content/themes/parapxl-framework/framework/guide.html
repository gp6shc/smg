The Parapxl Framework

The goal of this framework is to provide the developer with an easy to use tool set to quickly build out an informational website.

Classes

  • Boxes 100%
    • Creates meta boxes that hold custom fields
  • Columns 100%
    • Creates custom columns to display snippets, custom fields, and assigned taxonomy terms.
  • Formcast 85%
    • Allows for the easy creation of form elements.
  • Menus 100%
    • Creates the locations to assign menus.
  • Panel 95%
    • Creates a panel in wp-admin that allows you to create custom fields at the site level.
  • Post Types 100%
    • Creates custom post types to be used on the site.
  • Profile 100%
    • Creates custom fields for user profiles.
  • PXL 100%
    • Contains functions that Wordpress should implement to make developer's lives easier.
    • Also contains the pxlWalker class to help with css styling of menus that utilize drop-downs and fly-outs.
  • Resources 100%
    • Used to register and enqueue css styles and js scripts
  • Shortcodes 100%
    • Content Columns
    • Insert Custom Variable
    • List Posts
    • Sidebar Position
  • Sidebars 100%
    • Creates the dynamic sidebars.
  • Taxonomies 100%
    • Creates the custom taxonomies to be used with post types: post, page, custom post types.
    • Pass through to Post Types class.
  • Templates 100%
    • An extension of how Wordpress figures out which file to serve up as the template. We do it this way in order to have a more organized theme folder structure.
    • Types:
      • Default
      • Page Templates
      • Custom
  • Twitter 100%
    • User timeline [API Docs]
    • Search (e.g. Users, Hashtags, Terms, etc.) [API Docs | Search Operators]

Supporting Files

  • phpfn
    • A collection of homebrewed php specific functions that allow for shorter code.

    • A small php script for cropping, zooming and resizing web images (jpg, png, gif). Perfect for use on blogs and other applications.
